# 🌙 Nuit de la Femme Scientifique

## 📌 Description
Site web événementiel présentant "La Nuit de la Femme Scientifique", un événement académique visant à promouvoir la place des femmes dans les sciences et à encourager les jeunes filles à s’orienter vers les domaines scientifiques.

---

## 🎯 Objectifs du projet
- Valoriser les femmes dans les sciences
- Sensibiliser à l’égalité des chances
- Présenter un événement éducatif et inspirant
- Permettre l’inscription en ligne des participants

---

## 🛠️ Technologies utilisées
- HTML5
- CSS3
- Font Awesome (icônes)

---

## 📸 Aperçu
Projet composé de :
- Une page d’accueil avec présentation de l’événement
- Un programme détaillé
- Un formulaire d’inscription
- Une section contacts
